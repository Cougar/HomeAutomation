Using IntelliJ IDEA for Macbeth development

Installation

Download the most recent IntelliJ IDEA upgrade from http://www.intellij.com/download/idea/freeUpgrade. This is the full version, but it does of course require that you have a serial number, so keep your serial number ready the first time you start IDEA. The installation is very straight forward. Nothing particular to keep in mind there.

Basic Configuration

Now you need to configure IDEA to use the Java SDK you have installed on your system (if you don't have the SDK installed yet, just visit http://java.sun.com/ to download and install the latest J2SE SDK before going on!)

Open the Macbeth project file with IDEA. Then choose File->Project Properties. Go to the Paths-dialog and locate the drop-down list called "Target JDK". This is a list of the JDKs you have installed, and to get your installed Java SDK to show up here, you need to press the "..."-button and then press the "+"-button in the dialog box that shows up. Next, locate your J2SDK-directory (typically "C:\j2sdk1.4.2_03") and press OK until you're back to the Project Properties dialog. Now, in the drop-down list, choose the JDK you just configured and press OK.

IntelliJ IDEA is now ready for Macbeth development! You should be able to compile everything without any errors.

Code style

If you plan to participate in Macbeth development, you should make sure you follow the code style guidelines. Since IDEA does quite a lot of work for you while you're writing code, you might want to make sure IDEA follows these guidelines as well.
